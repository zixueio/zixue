"Copy element"，在 Chrome 开发者工具中，这是指将页面元素的 HTML 代码复制到剪贴板。

下面是在 Chrome 中复制元素的步骤：

  1. 右键点击网页上你想复制的元素，选择 "Inspect" 或 "Inspect Element"。这会打开 Chrome 的开发者工具，并且会自动定位到对应元素的 HTML 代码。

  2. 在开发者工具的 "Elements" 面板，右键点击你想要复制的元素的 HTML 代码。

  3. 在右键菜单中，你可以看到几个复制选项，包括 "Copy element"，这会复制元素的 HTML。你还可以选择 "Copy selector" 来复制 CSS 选择器，或者 "Copy XPath" 来复制 XPath。
